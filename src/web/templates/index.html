Bootstrap サンプルページ

はじめに

今回の課題では利用する技術スタックの制限はありません。フロントエンドフレームワークとして例えば Next.js や SvelteKit などを採用したいと思うなら、それらを利用することも可能です。私達としてはぶっちゃけ Amazon ECS の上で動く Dockerfile さえ作ってくれるなら問題ありません。ただ、それだとどうしても時間と手間が必要になるので、標準的な環境として Flask + Bootstrap によるベースも用意しておきました。今みているページは Flask + Bootstrap でページを作るときに使うかもしれない様々なパーツを配置したサンプルページです。実際に皆さんのアイデアを具現化していく際に、どんなパーツがあってどんなふうに表示されるのか、その参考にしてください。
もちろん、Bootstrap では表現の柔軟性が足りないという場合は自前で CSS を書いたり別のツールキットを採用したりしても構いません。とにかく全てを皆さんに委ねます。
